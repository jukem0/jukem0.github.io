Vi fik stiftet bekendsskab med basal termonology indenfor cybersikkerhed og hvordan man bedst kunne beskytte sig selv.

Nogle af de termer som vi lærte var:

  • Confidentiality, Integrity, og Availability (CIA):
    • Confidentiality: Kun dem som skal have adgang har adgang.
    • Integrity: Al information er korrekt samt afsender.
    • Availability: Hvornår er IT-systemet tilgængeligt? Hvordan beskytter man mod angreb? Kan det være slukket om natten?, Hvornår kan man opdatere det?
  • Passwords: De(t) kodeord man bruger til sine forskellige accounts, skal helst være unikt og noget som andre ikke kan gætte.
  • 2-factor-verification: En unik måde at identificerer sig selv når man skal ind på sin konto.
    • Noget man ved: Ens password
    • Noget man har: 2FA app, fx NemID-nøglekort
    • Noget man er: Ens øjne, fingeraftryk, ansigtsgenkendelse

Design af IT-Systemer

HTML står for HyperText Markup Language.

Man bruger HTML til at lave hjemmesider. HTML er det sprog computeren læser for at vise en hjemmeside, og det er i HTML filer at man definerer hvad der skal være på hjemmesiden.

HTML arbejder sammen med CSS og JavaScript til at vise hjemmesider, så de ser ordenligt ud.

✨Referencer:

  • HTML w3schools
  • Logbog skabelon

CSS står for Cascadin Style Sheet.

Man bruger CSS til at definere stilen på hjemmesiden, ved at enten vælge et helt element, en class, eller et id, og derefter definere hvad der skal ske med det.

CSS arbejder sammen med HTML hvis man linker dem sammen.

✨Referencer:

  • CSS w3schools
  • rgbA w3schools

De 5 principper er:

  • Contrast: Kontrasten imellem farverne
  • Alignment: Hvordan de forskellige elementer står i forhold til hinanden
  • Proximity: Hvor tæt de forskellige elementer er fra hinanden
  • Repetition: Dette princip går ud på at man skal bare være konsekvent med sine styles og bredder
  • White space: Hvor meget af siden der er tom, så hjemmesiden ikke ser rodet ud

✨Referencer:

  • 5 CORE PRINCIPLES IN GRAPHIC DESIGN - RULES YOU MUST OBEY

JavaScript er et sprog som bruges til at laves hjemmesider dynamiske.

Man kan fx bruge JavaScript (js) til at ændre indholdet på en hjemmeside ved at klikke på noget.

DOM står for Document Object Model, og det underbygger JavaScript.

✨Referencer:

  • DOM w3schools

Spiral modellen er en model som er inddelt i 3 forskellige faser hvorefter den gentager sigselv indtil man er tilfreds med sit produkt, Assess, Design, Build, repeat

  • Assess: Her finder du ud af hvad dine brugere har behov for, fx ved brug af spørgeskemaer, interviews, eller hjemmeside data
  • Design: Her designer du hvordan du vil have hjemmesiden til at se ud
  • Build: Her bygger du det som du har designet
  • (repeat): her gentager du fra assess fasen igen

✨Referencer:

  • Spiral model
